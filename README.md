# Deep Dive In Java Reflection

## 🎯 Goals

- 자바와 스프링에 대해 한층 더 깊은 이해를 하자.

1. 자바 프로그램의 실행방법과 JVM 아키텍처에 대해 배우자.
2. Reflection 를 이해하자.
3. Reflection 에 대한 이해를 바탕으로 Annotation 을 다시 공부해보자.
4. POJO 와 스프링의 철학에 대해 이해하자.
5. 스프링의 3대 핵심 기술에 대해서 공부하자.
